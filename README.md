# 🗃️ Product Inventory System

A console-based Product Inventory Management System built using Core Java.
Developed as a learning project to practice OOP, layered architecture, and Java fundamentals.

## 🚀 Features
- OTP-based login with mobile number validation
- Add, Remove, Search, and View products
- Update product price
- Auto-generated Product ID
- Colored console UI using ANSI escape codes

## 🏗️ Project Structure
- com.base            → Product model (POJO)
- com.business_layer  → Business logic
- com.validation_layer → Input validation & ID generation
- com.service_layer   → Main entry point (menu & login)

## 🛠️ Tech Used
- Java SE 21
- Core Java (OOP, Collections, Iterator, Exception Handling)

## ▶️ How to Run
1. Clone the repo
2. Open in Eclipse (or any Java IDE)
3. Run ProductMain.java
4. Login with a valid 10-digit mobile number starting with 6/7/8/9
5. Enter the OTP displayed on screen

## 📌 Note
This is a learning project built to strengthen Core Java fundamentals.
